# Gamestation JS

## Install

```js
<script src="gamestation-sdk.js" />
```

## API

### Data

```js
const account = window.data.account;
const score = window.data.score;
const items = window.data.items;
```

### Register score

```js
gameCenterSetScore();
```

### Buy item

```js
gameCenterBuyItem();
```
